# Real-Time Character Counter

## Description
A simple, interactive web app that counts characters in real-time as users type. Perfect for tracking input length, enforcing limits, and practicing JavaScript DOM manipulation.

## Features
- Real-time character count
- Maximum character limit (e.g., 50)
- User-friendly alerts when the limit is exceeded
- Responsive design with clean layout

## Technologies Used
- HTML
- CSS
- JavaScript
